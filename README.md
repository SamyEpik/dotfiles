# dotfiles

Dotfiles for my Arch Linux Hyprland rice

## Notes

- OS: Arch Linux x86_64
- Shell: zsh 5.9
- DE: Hyprland
- Terminal: kitty

## Key Features

- Modern and sleek design
- Pywal-based dynamic theming
- TUI focused look
